    /submit

    /submit post

    post

    Run Cello!

    post /submit

    Run Cello!

    • Request

    Query Parameters

    • jobid: required (string)

      job name (unique)

    • verilog: required (string)

      Verilog text (not filepath)

    • inputs: required (string)

      Input promoter data. Rows of space-separated name, low, high, dnaseq

      Example:

      pTac 0.0034 2.8 AACGATCGTTGGCTGTGTTGACAATTAATCATCGGCTCGTATAATGTGTGGAATTGTGAGCGCTCACAATT
    • outputs: required (string)

      Output gene data. Rows of space-separated name, dnaseq (each dnaseq should concatenate ribozyme, rbs, cds, terminator)

      Example:

      YFP CTGAAGCTGTCACCGGATGTGCTTTCCGGTCTGATGAGTCCGTGAGGACGAAAC
    • user_options: required (string)

      custom flags, not intended for new users. Space + dash separated strings.

      Example:

      -figures false -plasmid false -assignment_algorithm hill_climbing
